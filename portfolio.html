GameR Game Review Website

Dec. 2016

GameR is a group website project that I worked with three other classmates. We each decided that we were going to create a single page and connect them when we were done; I created the Game Review page. We were tasked to create a themed uniform website using all of the techniques we learned in class. Before we began, we came up with a general design of the website so each page could would have uniformity. To create this website we used Bootstrap, jQuery, and Javascript. Formy page, I created buttons that would show the review of Battlefield 1 and Destiny. I also used buttons which showed pictures from each games' respective subreddits.
